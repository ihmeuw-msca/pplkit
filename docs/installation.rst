============
Installation
============

The package :code:`pplkit` is written in Python and requires Python 3.12 or later.

From PyPI
---------

.. code::

   pip install pplkit

For Developers
--------------

Clone the repository and install the package in development mode with test and
docs extras:

.. code::

   git clone https://github.com/ihmeuw-msca/pplkit.git
   cd pplkit
   pip install -e ".[test,docs]"
